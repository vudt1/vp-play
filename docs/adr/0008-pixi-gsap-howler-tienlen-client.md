# PixiJS v8 + GSAP 3 + Howler for Tiến Lên client (offline LAN)

The Tiến Lên Module client moves from a DOM + Phaser stub to a full-canvas **PixiJS v8** stage (base resolution 1920×1080 with letterbox/pillarbox), **GSAP 3** for deal/play animations, and **Howler.js** for SFX. Vendor min builds ship under `public/modules/tienlen/js/libs/` (no CDN). Socket.IO client continues to load from the app server path (`APP_PREFIX` + `/socket.io/socket.io.js`) so client and server versions stay matched on the LAN. Card faces stay the existing SVG set; game rules and deal authority remain on the server (`src/modules/tienlen/domain/`, `src/modules/tienlen/rooms/roomTable.js`).
