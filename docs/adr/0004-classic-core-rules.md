# Classic core Tiến Lên rules only

MVP implements legal combos, same-shape beat, in-turn specials (tứ quý / 3 đôi thông vs single 2; 4 đôi thông vs single|pair 2 or tứ quý), every hand opens with 3♠ holder, full finish ranking, and simple finish-order points.

Explicit non-goals: tới trắng, đền bài, thối 3 bích, cóng multipliers, AI autoplay, out-of-turn chặt, ELO. Full research remains in `spec/logic_game.md` for a later expansion decision.
